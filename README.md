# Tip Calculator App

A responsive tip calculator that helps you calculate the correct tip amount and the total cost of the bill per person.

## Features

- Calculate the **tip amount per person**
- Calculate the **total amount per person**
- Responsive layout (mobile / tablet / desktop)
  
## Built With
- HTML
- CSS
- JavaScript
  
## How to Use

1. Enter the bill amount
2. Select a tip percentage (or enter a custom tip if available)
3. Enter the number of people
4. View:
   - Tip amount per person
   - Total amount per person
